# 团队销售报表 KPI 基准

本文件用于团队日报、团队周报、团队月报中的管理判断。只服务于销售管理者团队视角，不用于单个 AE 的个人日报、周报、月报。

## 使用原则

- KPI 只作为预警参考，不替代管理判断。
- 团队平均必须和个体异常同时看，不能互相掩盖。
- 数据缺失时标注“待确认”，不要把缺失 AE 排除后直接得出健康结论。
- 指标异常必须说明影响对象、影响金额或管理动作。

## 团队日报关注指标

| 指标 | 健康信号 | 预警信号 | 管理含义 |
|------|----------|----------|----------|
| 今日成交/回款 | 有关键结果或按计划推进 | 关键成交或回款延期 | 需判断是否影响本周目标 |
| 今日关键商机推进 | 高价值商机有明确下一步 | 关键商机会议取消或无结论 | 可能需要主管当天介入 |
| AE 当日有效跟进 | 关键 AE 有有效动作 | 关键 AE 无有效跟进记录 | 需确认是否数据缺失或执行异常 |
| P0 完成情况 | P0 当天完成或有明确原因 | P0 未完成且影响客户节点 | 次日管理动作必须覆盖 |
| 新增风险 | 风险可控且有负责人 | 新增高风险客户/竞品/关键人失联 | 进入主管 P0 或 P1 |

## 团队周报关注指标

| 指标 | 健康值 | 预警线 | 说明 |
|------|--------|--------|------|
| 团队目标完成率 | >=80% | <60% | 低于预警线需解释缺口来源 |
| 团队 Pipeline 覆盖 | Pipeline >= 月目标 x 3 | < 月目标 x 2.5 | 覆盖不足影响后续达成 |
| 深度阶段商机占比 | >=30% | <20% | 深度阶段包括 POC、方案、报价、合同 |
| 周成交/回款达成 | 符合周节奏 | 连续两周低于节奏 | 可能影响月目标 |
| 逾期任务占比 | <=10% | >20% | 团队执行风险 |
| 高风险商机数 | 可控且有动作 | 连续增加或无改善 | 需要主管资源分配 |

## 团队月报关注指标

| 指标 | 健康值 | 预警线 | 说明 |
|------|--------|--------|------|
| 月目标完成率 | >=80% | <60% | 低于预警线需做经营复盘 |
| 月回款完成率 | >=80% | <60% | 区分成交与实际回款 |
| Pipeline 新增/消耗比 | >=1.5 | <1 | 小于 1 表示存量被消耗 |
| 深度商机增长 | 增长或稳定 | 下降 | 下月成交基础不足 |
| AE 表现分化 | 少量波动 | 多人持续异常 | 需要管理动作 |
| 高风险客户占比 | <=20% | >30% | 团队风险敞口偏高 |

## AE 横向异常阈值

| 异常 | 触发条件 | 管理判断 |
|------|----------|----------|
| 目标落后 | 单个 AE 低于团队平均 20% 以上 | 关注原因，不直接定性能力问题 |
| 持续落后 | 连续 2 期目标完成率低于 50% | 需要主管复盘 |
| Pipeline 集中 | 单一客户占某 AE Pipeline 40% 以上 | 单点风险 |
| 执行异常 | 单个 AE 逾期任务 >2 个 | 检查承诺管理和客户阻塞 |
| 风险延续 | 同一高风险商机连续 2 期未改善 | 需要升级处理 |
| 数据缺失 | 缺少关键 CRM 或个人报表数据 | 先标注缺口，不做绩效定性 |

## Pipeline 健康度

阶段分布参考：

```text
初步接触：<=30%
需求确认：20%-30%
POC/方案：20%-30%
合同谈判：15%-25%
已成交/赢单：结合周期看
```

预警：

- 初步接触 >40%：Pipeline 偏浅。
- 深度阶段 <20%：未来成交支撑不足。
- Pipeline 增长 20% 以上但成交率下降：增长质量存疑。
- 某行业占团队 Pipeline 40% 以上：行业集中风险。
- 某 AE 占团队 Pipeline 50% 以上：资源集中风险。

## 风险评级

| 等级 | 说明 | 主管动作 |
|------|------|----------|
| 低 | 正常推进，无明显管理动作 | 观察 |
| 中 | 有风险信号，需要明确责任人 | 跟进 |
| 高 | 影响目标、关键客户或多个 AE | 主管介入 |
| 待确认 | 数据不足或证据冲突 | 补数据后判断 |

风险升级参考：

- 进度停滞 2 期：中风险。
- 进度停滞 3 期：高风险，建议主管介入。
- 竞品涉及 2 个客户：中风险。
- 竞品涉及 3 个以上客户或多个 AE：高风险。
- Champion 失联 2 期：中风险。
- Champion 失联 3 期：高风险。

## 团队健康度评分

可使用以下权重作为参考：

```text
团队健康度 = 0.30 x 目标完成得分
          + 0.25 x Pipeline质量得分
          + 0.20 x 成交/回款节奏得分
          + 0.15 x 风险控制得分
          + 0.10 x 执行质量得分
```

等级：

| 分值 | 等级 | 说明 |
|------|------|------|
| 80-100 | 良好 | 关键指标正常，风险可控 |
| 60-79 | 关注 | 存在 1-2 个管理问题 |
| 0-59 | 危险 | 多项指标异常或高风险集中 |
| 无法计算 | 待确认 | 数据缺失严重 |

## 组合预警模式

| 触发条件 | 预警 | 可能问题 |
|----------|------|----------|
| 目标完成低 + Pipeline 覆盖不足 | 高 | 当期和后续都承压 |
| Pipeline 增长 + 深度商机下降 | 中 | Pipeline 虚高 |
| 多个 AE 同行业竞品信号 | 高 | 竞品集中渗透 |
| 高风险客户集中在一个 AE | 中高 | 资源或方法需要介入 |
| 逾期率上升 + 承诺兑现率下降 | 中 | 执行节奏或承诺管理问题 |
| 成交金额达标 + 回款不足 | 中 | 商务或交付回款风险 |
