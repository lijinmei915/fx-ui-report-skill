# 个人销售报表 KPI 基准

本文件用于个人日报、周报、月报中的指标判断。只服务于单个销售人员 / AE / 客户经理，不用于团队汇总、AE 排名或主管管理看板。

## 使用原则

- KPI 只作为预警参考，不替代业务判断。
- 同一指标要结合行业、客户规模、销售阶段和数据来源解释。
- 如果数据缺失，标注“无法判断”，不要用平均值补齐。
- 当 CRM 数据、销售人员自述和 `JournalObj` 历史日志冲突时，以 CRM 事实优先，并标注冲突。
- 本地 `memory/` 仅作兜底参考，不覆盖 CRM 或 `JournalObj`。

## 日报关注指标

| 指标 | 健康信号 | 预警信号 | 说明 |
|------|----------|----------|------|
| 当日关键结果 | 至少有明确客户反馈、阶段推进、交付物或成交之一 | 只有动作，没有结论 | 日报看结果，不看流水账 |
| 当日 P0 完成率 | 100% | 任一 P0 未完成且无原因 | P0 未完成必须进入次日计划 |
| 关键客户响应 | 有明确回复或下一步 | 关键客户当天未回复 | 对临近成交客户尤其敏感 |
| 沟通结论率 | 大部分跟进有结论 | 多条“沟通了/推进中”无结论 | 触发假推进识别 |
| 明日计划质量 | 有客户、联系人、动作、目标 | 只有“继续跟进” | 行动不可验证时需重写 |

## 周报关注指标

| 指标 | 健康值 | 预警线 | 说明 |
|------|--------|--------|------|
| 周新增客户 | 2-3 个 | <1 个/周 | 低于预警线说明客户储备不足 |
| 周跟进客户 | 8-15 个 | <5 个 | 跟进量不足会影响成交 |
| 周待办完成率 | >=80% | <60% | 逾期/延期过多说明执行质量差 |
| 周 Pipeline 新增 | >=月目标的 10% | <5% | Pipeline 补充不足 |
| 连续未推进商机 | <=2 个 | >3 个 | 2 周以上无实质进展需预警 |
| 周成交或阶段推进 | 有成交或关键阶段推进 | 无成交且无深度推进 | 需判断 Pipeline 是否虚高 |

## 月报关注指标

| 指标 | 健康值 | 预警线 | 说明 |
|------|--------|--------|------|
| 月目标完成率 | >=80% | <60% | 低于预警线需解释缺口来源 |
| 月成交率 | 15%-25% | <10% | 成交数 / 有效商机数 |
| 月 Pipeline 覆盖 | Pipeline >= 月目标 x 3 | < 月目标 x 2.5 | 覆盖不足会影响下月 |
| 深度阶段商机占比 | >=30% | <20% | 深度阶段包括 POC、方案、报价、合同 |
| Pipeline 新增/减少比 | >=1.5 | <1 | 小于 1 表示存量被消耗 |
| 月度承诺兑现率 | >=80% | <60% | 连续低于预警线说明执行或承诺管理有问题 |

## Pipeline 健康判断

| 指标 | 健康信号 | 预警信号 |
|------|----------|----------|
| Pipeline 充足率 | Pipeline >= 月目标 x 3 | < 月目标 x 2.5 |
| 阶段分布 | 深度阶段 >=30% | 初步接触 >40% |
| 成交临近度 | 有明确签约时间和决策流程 | 大额商机长期无 EB、预算或流程 |
| 金额变化解释 | 新增、升阶、成交、丢单都有原因 | 金额变化但原因缺失 |
| 赢率可信度 | 赢率有客户行为支撑 | 自评高但无客户推进证据 |

阶段分布参考：

```text
初步接触    25%
需求确认    35%
方案报价    20%
合同签订    10%
已成交      10%
```

如果初步接触占比超过 40%，标注 Pipeline 偏浅。  
如果 Pipeline 上升但深度阶段没有增加，标注增长质量存疑。

## 销售周期基准

| 行业 | 平均成交周期 | 快速成交 | 异常偏长 |
|------|--------------|----------|----------|
| ICT/高科技 | 60-90 天 | <45 天 | >120 天 |
| 制造业 | 75-120 天 | <60 天 | >150 天 |
| 消费品 | 45-75 天 | <30 天 | >90 天 |
| 医疗健康 | 90-180 天 | <60 天 | >240 天 |
| 教育培训 | 20-45 天 | <14 天 | >60 天 |

销售周期偏长时，结合以下信号判断风险：

- 决策流程是否明确
- EB 是否确认
- Champion 是否仍活跃
- 预算是否确认
- 是否出现竞品
- 是否存在连续延期

## 待办完成率基准

| 指标 | 健康值 | 预警线 |
|------|--------|--------|
| 应完成/实际完成 | >=80% | <60% |
| 逾期任务占比 | <=10% | >20% |
| 延期任务占比 | <=20% | >30% |
| 延期原因有效率 | >=70% | <50% |
| 连续延期事项 | 0-1 个 | >=2 个 |

延期原因必须可验证。  
“客户忙、下周再说、内部还在看”连续出现时，进入承诺泡沫或假推进判断。

## 竞品风险基准

| 竞品渗透阶段 | 时间窗口 | 行动紧迫度 |
|--------------|----------|------------|
| 客户提及竞品 | 4-6 周 | 关注 |
| 竞品已 Demo | 2-4 周 | 中 |
| 竞品已 POC | 1-2 周 | 高 |
| 竞品已签约 | 无 | 战略复盘 |

出现竞品后，个人报表必须回答：

- 竞品是否确认存在，还是推断
- 竞品进入到哪个阶段
- 我方当前落后在哪里
- 下一步验证或反击动作是什么

## MEDDIC 风险参考

| 缺失要素 | 风险等级 | 影响 |
|----------|----------|------|
| 无 Metrics | 中 | 不知道客户为什么必须买 |
| 无 Economic Buyer | 高 | 不知道真正决策人是谁 |
| 无 Decision Criteria | 中 | 可能被竞品标准牵引 |
| 无 Decision Process | 中 | 决策周期和关键节点不可控 |
| 无 Pain 明确证据 | 中 | 需求可能只是口头兴趣 |
| 无 Champion | 中高 | 客户内部无人推动 |
| 多个要素缺失 | 高 | 商机成熟度不足，不应高估赢率 |

Champion 活跃度参考：

| 状态 | 正常表现 | 预警信号 |
|------|----------|----------|
| 活跃 | 每周至少一次有效互动 | 连续 1 周无联系 |
| 强支持 | 主动安排内部会议或推进材料 | 突然沉默 |
| 弱支持 | 偶尔回复但不推动内部 | 连续 2 周无实质动作 |
| 口头支持 | 只说“好、可以、再看看” | 无下一步、无内部推动 |

## 组合预警模式

| 触发条件 | 预警 | 可能问题 |
|----------|------|----------|
| Pipeline 下降 + 待办完成率低 | 高 | 执行质量和储备同时出问题 |
| Pipeline 上升 + 无深度推进 | 中 | Pipeline 虚高 |
| 新增客户减少 + 跟进客户稳定 | 中 | 客户储备不足 |
| 成交率低 + 成交周期拉长 | 中高 | 客户质量或推进方式有问题 |
| 延期任务 >30% | 中 | 承诺过度或客户真实意向不足 |
| Champion 失联 + 决策周期拉长 | 高 | 决策链变化或竞品介入 |
